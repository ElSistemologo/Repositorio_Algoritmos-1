En esta práctica de Laboratorio se vió una breve introducción acerca de la sintaxis, semántica y pragmática
del lenguaje de programación Python, incluyendo temáticas tales como:
1. Python: Data Types and Data Structures
2. Python: Logical Operations and Loop
3. Functions and Objective-Oriented Programming
4. NumPy and Basic Pandas
5. Pandas-Resampling and DataFrame
En consecuencia, se encuentran ejemplos prácticos de cada uno de los temas enunciados, con sus respectivos comentarios.
